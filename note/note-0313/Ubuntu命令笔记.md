#### 终端字体大小

增大字体：`ctrl`+`shift`+`"+"`

减小字体：`ctrl`+`"-"`

#### 常用命令

`ls`： list简写，看到当前目录下有那些文档

`ll`：显示隐藏的文件夹，所有者，权限，创建日期

`pwd`：print working directory 简写，显示工作目录的路径名称

`mkdir`：新建目录

`↑`：方向键"上"，复制上一行

`cd`：切换目录

`tab`：自动补齐

`nano`：新建文字档案

`cp`：copy简写，`cp <要拷贝的文件> ../目标路径`

`rm`：删除 `rm -rf`强制删除

`cat`：查看内容

`head -n <num>`：查看前面num行

`tail -n <num>`：查看后面num行

`bc`：计算器

`date`：显示日期

`cal`：显示日历
